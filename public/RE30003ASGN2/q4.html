Part b: X-bar and R Control Charts for Phase I ¶

Objective: Create an X-bar and R control chart for the Phase I data to establish baseline control limits for the mean and range of piston ring diameters. Identify any points that fall outside the control limits in Phase I and interpret whether any adjustments might be necessary to bring the process fully under control.

Understanding X-bar and R Control Charts¶

  • X-bar Chart: Monitors the process mean over time.
  • R Chart: Monitors the process variability (range) over time.

Key Components:

  • Centerline (CL): The average of the sample means (for X-bar) or sample ranges (for R).
  • Upper Control Limit (UCL): CL + (A2 * R̄) for X-bar; CL + D4 * R̄ for R chart.
  • Lower Control Limit (LCL): CL - (A2 * R̄) for X-bar; CL - D3 * R̄ for R chart.

Constants for X-bar and R Charts (for sample size n=5):

Chart Type A2 D3 D4
X-bar 0.577 N/A N/A
R N/A 0 2.114

Note: D3 is zero for n=5.

Source: Statistical Process Control (SPC) tables.

# Constants for sample size n=5
A2 = 0.577
D3 = 0
D4 = 2.114

# Calculate sample ranges for Phase I
phase_I = phase_I.copy()
phase_I['range'] = phase_I.groupby('sample')['diameter'].transform(lambda x: x.max() - x.min())

# Calculate average of sample means and ranges
X_bar = phase_I.groupby('sample')['diameter'].mean().mean()
R_bar = phase_I['range'].mean()

Part d: Process Capability Assessment ¶

Objective: Assess the process capability for the piston ring diameters, assuming a target diameter of 74 mm and a tolerance of ±0.030 mm. Calculate the Process Capability Index (Cp) based on Phase I data. Comment on whether the process is capable of consistently producing piston rings within the specified tolerance, and discuss any adjustments needed if capability is insufficient.

Understanding Process Capability Indices¶

  • Cp (Process Capability): Measures the potential capability of a process assuming it is centered between the specification limits.

$$ Cp = \frac{USL - LSL}{6\sigma} $$

  • Cpk (Process Capability Index): Takes into account the process mean's deviation from the target.

$$ Cpk = \min\left(\frac{USL - \mu}{3\sigma}, \frac{\mu - LSL}{3\sigma}\right) $$

Where:

  • USL: Upper Specification Limit
  • LSL: Lower Specification Limit
  • σ: Standard deviation of the process
  • μ: Mean of the process

Interpretation:

  • Cp, Cpk ≥ 1.33: Generally considered capable.
  • Cp, Cpk ≥ 1.67: Highly capable.
  • Cp, Cpk < 1.33: Process may require improvement.
# Define specification limits
USL = 74.030  # Upper Specification Limit (74 mm + 0.030 mm)
LSL = 73.970  # Lower Specification Limit (74 mm - 0.030 mm)

# Calculate process standard deviation (σ) from Phase I data
sigma = phase_I_summary['mean_diameter'].std()

# Calculate process mean (μ) from Phase I data
mu = X_bar

# Calculate Cp
Cp = (USL - LSL) / (6 * sigma)

# Calculate Cpk
Cpk_upper = (USL - mu) / (3 * sigma)
Cpk_lower = (mu - LSL) / (3 * sigma)
Cpk = min(Cpk_upper, Cpk_lower)

# Display the results

print(f"Upper Specification Limit (USL): {USL:.3f} mm")

print(f"Process Capability Index (Cp): {Cp:.3f}")
print(f"Process Capability Index (Cpk): {Cpk:.3f}")
